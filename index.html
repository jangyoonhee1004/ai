📇 AI 용어 카드

꼭 알아야 할 10가지 핵심 용어

1️⃣ 프롬프트 (Prompt)

의미: AI에게 보내는 질문이나 명령어

예시: "회의록을 요약해줘" ← 이것이 프롬프트

2️⃣ ChatGPT

의미: 대화형 AI 서비스 (챗봇의 일종)

비유: 책을 다 읽어본 똑똑한 비서

3️⃣ 맥락 (Context)

의미: AI가 답변을 이해하기 위해 필요한 배경 정보

예시: "저는 직장인이고, 업무 보고서를 써야 해요" ← 맥락 제공

4️⃣ 토큰 (Token)

의미: AI가 이해하는 텍스트의 단위 (대략 단어 하나)

왜 중요?: 한 번에 처리할 수 있는 토큰 수가 제한되어 있음

5️⃣ 생성형 AI (Generative AI)

의미: 새로운 콘텐츠를 만들어내는 AI

예시: 글쓰기, 그림 그리기, 코드 작성 등

6️⃣ 프롬프트 엔지니어링

의미: AI에게 효과적으로 질문하는 기술

비유: AI와 잘 대화하는 방법

7️⃣ 할루시네이션 (Hallucination)

의미: AI가 사실이 아닌 정보를 그럴듯하게 만들어내는 현상

주의: 중요한 정보는 반드시 검증이 필요해요

8️⃣ 온도 (Temperature)

의미: AI 답변의 창의성 정도를 조절하는 설정

낮으면: 정확하고 일관된 답변 / 높으면: 창의적이고 다양한 답변

9️⃣ 퓨샷 러닝 (Few-shot Learning)

의미: 몇 가지 예시를 보여주고 패턴을 학습시키는 방법

예시: "이런 형식으로 써줘" 하고 예시 2-3개 제공

🔟 체인 (Chain)

의미: 여러 단계의 프롬프트를 연결해서 사용하는 것

예시: 1단계 요약 → 2단계 분석 → 3단계 제안

💬 질문하는 3가지 방법

상황에 맞는 질문 방식 선택하기

🟢 방법 1: 직접 질문 (가장 기본)

언제: 간단한 정보가 필요할 때

• "프로젝트 관리 도구 추천해줘"
• "마케팅 전략이 뭐야?"
• "Python으로 데이터 분석하는 방법 알려줘"

🟡 방법 2: 맥락 + 질문 (추천)

언제: 나에게 맞는 구체적인 답이 필요할 때

나는 [상황]이야.
[문제/목표]가 있는데,
[원하는 것]을 알려줘.

예시:

나는 신입사원 교육을 담당하는 직원이야.
교육 자료가 너무 이론적이라 학습자들이 지루해하는 문제가 있는데,
실습 중심의 참여형 교육 방법을 알려줘.

🔴 방법 3: 역할 부여 + 질문 (고급)

언제: 전문가의 관점이 필요할 때

너는 [전문가]야.
[상황]에 대해
[관점]으로 [원하는 것]을 해줘.

예시:

너는 20년 경력의 마케팅 전문가야.
신제품 출시 전략에 대해
데이터 기반 관점으로 분석하고 실행 가능한 방안을 3가지 제안해줘.
💡 팁: 처음엔 방법 1로 시작하고, 답변이 마음에 안 들면 방법 2나 3으로 다시 질문해보세요!

🎓 고급 프롬프트 기법

전문가처럼 AI를 다루는 핵심 테크닉

🔗 1. 프롬프트 체이닝 (Chaining)

설명: 큰 작업을 여러 단계로 쪼개서, 각 단계의 결과를 다음 단계의 입력으로 사용

언제 사용: 보고서, 기획서, 복잡한 분석

1단계: "다음 데이터의 핵심 패턴 3가지를 찾아줘: [데이터]"
→ AI가 패턴 3가지 제시

2단계: "위에서 찾은 첫 번째 패턴에 대해, 원인과 시사점을 분석해줘"
→ AI가 깊이 있는 분석 제공

3단계: "이 분석을 바탕으로 경영진 보고서 초안을 작성해줘. 분량 2페이지, 전문적인 톤"

🎭 2. 페르소나 활용 (Role Playing)

설명: AI에게 특정 전문가의 역할을 맡겨서, 그 관점에서 답변하게 하는 방법

언제 사용: 다각도 검토, 중요한 의사결정

마케터 역할: "너는 20년 경력의 마케팅 전문가야. 신제품 출시 전략을 고객 관점에서 평가해줘."

재무 담당자 역할: "너는 재무 담당자야. 같은 전략을 비용과 ROI 관점에서 검토해줘."

리스크 관리자 역할: "너는 리스크 관리 전문가야. 이 전략의 잠재적 위험 요소를 찾아줘."

🎯 3. 제약 조건 설정

설명: "이것만 해줘", "이건 하지 마" 같은 명확한 경계를 설정

"직원 복지 향상 아이디어를 10가지 제안해줘.

제약 조건:
- 예산: 1인당 월 5만원 이내
- 실행 기간: 3개월 이내 가능한 것
- 제외: 금전 지급, 업무시간 단축 관련
- 포함: 직원이 직접 체감할 수 있는 것
- 형식: 표로 정리 (아이디어명, 예상 비용, 실행 난이도)"
